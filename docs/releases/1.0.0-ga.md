## 🚀 Highlights

* **Executable proof included** — reproducible **GREEN → RED → GREEN** loop using **build-time guardrails only**.
* **Architecture guardrails (default-enabled)** — generated **ArchUnit** tests fail the build **deterministically** on drift; configurable via `--guardrails`.
* **Two architecture models** — `standard` is the default layout; `hexagonal` can be selected explicitly via `--layout`.
* **Buildable-by-default output** — generated projects pass **`mvn verify`** immediately after generation.
* **GA contracts are explicit** — guarantees, release discipline, and non-goals are documented.

---

## ✅ GA guarantees (1.0.0)

* **Deterministic generation** — same input → same output.
* **Test-ready output** — generated projects pass **`mvn verify`** immediately.
* **Layout choice** — generated projects default to `standard`; `hexagonal` can be selected explicitly via `--layout`.
* **Guardrails (default-enabled)** — generated ArchUnit tests evaluated at **build time** (`mvn verify`), configurable via `--guardrails` (`none` | `basic` | `strict`).
* **Clean core by construction** — generator keeps its **domain** framework-free (self-guarded by CI).

## 📦 Output contract (generated project)

Generated projects are **single-module** and include:

* `pom.xml` (with Maven Wrapper)

  * `.mvn/wrapper/maven-wrapper.properties`
  * `mvnw`
  * `mvnw.cmd`
* `src/main/java/<basePackage>/...`
* `src/test/java/<basePackage>/...`
* `src/main/resources/application.yml`
* `.gitignore`
* `README.md` (part of the generated contract surface)

> The generated README and the generated ArchUnit tests are part of the **delivered contract surface**.

## 🧾 Release & compatibility discipline

Versions are **contracts**:

* **MAJOR** — contract reset
* **MINOR** — backward-compatible expansion
* **PATCH** — bug fixes only

Patch releases in `1.0.x` will **not**:

* change CLI contracts
* modify generated project structure
* alter guardrails semantics

Reference:

* [Release Discipline](https://github.com/blueprint-platform/codegen-blueprint/blob/main/docs/policies/release-discipline.md)
